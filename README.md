# quantum-projects
Basic quantum simulation using qiskit
This repository contains implementations of fundamental quantum computing algorithms and circuits using Python and Qiskit. These projects demonstrate core concepts such as superposition, entanglement, and quantum measurement.


## Algorithms & Circuits Implemented
### 1. Deutsch-Jozsa Algorithm
- Determines whether a function is constant or balanced  
- Demonstrates exponential speedup over classical approach

### 2. Bernstein-Vazirani Algorithm
- Finds a hidden binary string with a single query  
- Shows efficiency of quantum query models

### 3. Grover's Algorithm
- Searches an unsorted database with quadratic speedup  
- Implemented for small qubit systems using Qiskit  

### 4. Quantum Entanglement Circuit
- Creation of entangled states (Bell states)  
- Demonstrates non-classical correlations between qubits

### 5. Statevector Probability Calculation
- Computation of measurement probabilities from quantum statevectors  
- Analysis of quantum states using Qiskit tools
